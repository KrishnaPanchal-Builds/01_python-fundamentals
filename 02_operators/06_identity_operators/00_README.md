# 🆔 06 - Identity Operators

> **Module 02 • Operators**

Identity operators are used to determine whether two variables refer to the **same object in memory**. Unlike comparison operators, which compare values, identity operators compare the identity of objects.

These operators return a Boolean value (`True` or `False`) and are commonly used when checking whether two variables reference the exact same object.

# 🔍 Identity vs Equality

Identity operators and comparison operators are often confused.

| Identity (`is`) | Equality (`==`) |
|-----------------|-----------------|
| Compares whether two variables refer to the same object. | Compares whether two values are equal. |
| Returns `True` only if both variables point to the same object. | Returns `True` if the values are equal, even if they are different objects. |

# 📌 Key Takeaways

- Identity operators compare object identity, not values.
- `is` checks whether two variables refer to the same object.
- `is not` checks whether two variables refer to different objects.
- Identity operators always return a Boolean value.
- Do not use `is` as a replacement for `==` when comparing values.
