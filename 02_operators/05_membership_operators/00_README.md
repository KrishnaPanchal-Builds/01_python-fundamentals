# 👥 07 - Membership Operators

> **Module 02 • Operators**

Membership operators are used to determine whether a value exists inside a collection such as a string, list, tuple, set, or dictionary.

These operators return a Boolean value (`True` or `False`) and are commonly used for searching, validation, filtering, and access control in Python programs.

# 📚 Operators Covered

| Operator | Description | Example |
|----------|-------------|---------|
| `in` | Returns `True` if the value exists in a collection | `"Python" in skills` |
| `not in` | Returns `True` if the value does not exist in a collection | `"Java" not in skills` |
