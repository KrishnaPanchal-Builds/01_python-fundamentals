# 🔗 03 - Logical Operators

> **Module 02 • Operators**

Logical operators are used to combine or modify multiple conditions in Python. They work with Boolean (`True` and `False`) values and are essential for building intelligent, decision-making programs.

Logical operators allow programs to evaluate multiple comparisons together and determine whether one or more conditions are satisfied.

# 📚 Operators Covered

| Operator | Description | Example |
|----------|-------------|---------|
| `and` | Returns `True` if both conditions are `True` | `(10 > 5) and (8 < 12)` |
| `or` | Returns `True` if at least one condition is `True` | `(10 < 5) or (8 < 12)` |
| `not` | Reverses the Boolean result | `not (10 > 5)` |
